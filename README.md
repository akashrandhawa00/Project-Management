# Project Manager App

A simple project and task management application built with React. This app allows users to create projects, assign tasks to them, and manage everything through a clean sidebar-based interface.

---

## Features

- Create and delete projects
- Select a project from a sidebar list
- Add tasks to individual projects
- Delete tasks from projects
- Input validation with modal feedback
- Dynamic UI with conditional rendering (no project / new project / selected project views)

---

## Tech Stack

- React (Functional Components)
- React Hooks (`useState`, `useRef`)
- Tailwind CSS for styling
- Component-based architecture
